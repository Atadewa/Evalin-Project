Anda sedang offline

Aplikasi ini mendukung mode offline sebagian. Anda dapat melanjutkan mengerjakan soal dan jawaban akan tersimpan sementara di perangkat. Ketika kembali online, draft akan disinkronkan otomatis.

Silakan kembali ke dashboard atau lanjutkan ujian. Jika butuh, muat ulang setelah koneksi internet tersedia.

Pastikan membuka aplikasi melalui HTTPS atau localhost untuk pengalaman PWA penuh.
